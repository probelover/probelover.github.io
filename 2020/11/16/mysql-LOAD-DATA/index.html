mysql LOAD DATA

MySQL 中提供了LOAD DATA INFILE语句来插入数据。将该文件中的数据插入到当前数据库的表中。

语法如下：

LOAD DATA LOCAL INFILE ‘file_name’ INTO TABLE table_name
FIELDS TERMINATED BY ‘:’
LINES TERMINATED BY ‘\r\n’;

服务端需要开启相关权限：

查看权限是否开启：
show variables like ‘local_infile’;
设置状态开启：
set global local_infile = 1;

同时在使用命令进行此操作时，需要加上相关参数

–local-infile=1
如：mysql –local-infile=1 -uuser -ppwd
